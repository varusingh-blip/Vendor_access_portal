# Vendor Access Portal

## Overview

Vendor Access Portal is a lightweight, centralized web dashboard designed to provide quick and structured access to WAN, Cloud, Colocation, Hardware, and Internal infrastructure vendor portals.

This tool consolidates multi-vendor access into a single, clean enterprise interface for Network Operations and Infrastructure teams.

The portal is built as a static HTML application and hosted via GitHub Pages for easy access and zero maintenance overhead.

---

## Categories

The dashboard is organized into the following sections:

### WAN / ISP
- GTT Envision
- Arelion Portal
- Lumen Control Center
- Zayo Tranzact
- Telstra Support

### Cloud
- AWS SSO
- Microsoft Azure
- Oracle Cloud
- Oracle Support

### Colocation
- CoreSite Portal

### Hardware
- Arista
- Arista cVaaS
- Juniper Support
- Fortinet Support

### Internal Tools
- NVIDIA NetQ
- GDCO (Azure WVD)
- Cumulus Support

---

## Features

- Category-based navigation
- Real-time search filtering
- Dark mode toggle
- Responsive design
- Lightweight static deployment
- Zero backend dependencies

---

## Usage

1. Open the hosted URL.
2. Use the left sidebar to navigate categories.
3. Use the search bar to quickly filter vendors.
4. Toggle Dark Mode if preferred.
5. Click any vendor card to open the portal in a new tab.

## Security Note

This portal does not store credentials or sensitive data.  
It only provides structured access to external vendor login portals.

Authentication is handled by each respective vendor platform.
